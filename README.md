# Subscription Tracker & Renewal Dashboard

## Tech Stack
- React + Vite + Tailwind CSS
- Node.js + Express
- In-memory storage

## Features
- Add subscriptions
- Monthly burn rate calculation
- Upcoming renewal alerts
- Active/Paused toggle
- Responsive dashboard

## Run Backend

```bash
cd backend
npm install
npm run dev
```

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```
